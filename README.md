# AETHER & HERMES - Linux Driver Projects

## Projects Overview
1. **AETHER**: Software Defined Radio Simulator
2. **HERMES**: Hardware Security Module

## Hardware
Using 37-in-1 Arduino sensor kit with Raspberry Pi.

## Development
- Kernel: 6.1.0-rpi7-rpi-v8
- Compiler: GCC
- Platform: Raspberry Pi (ARM64)
